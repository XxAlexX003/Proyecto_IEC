Calculadora de Interés Simple

Fórmulas base (con tiempo en años):

M = C (1 + i · t) | I = C · i · t | i = (M / C − 1) / t | t = (M / C − 1) / i
Los campos bloqueados son los que el programa calcula por ti.
%
Ej: si dice “1.5% mensual”, escribe 1.5 y elige “% mensual”.
Llena solo lo que necesites. Ej: 1 año y medio → 1 año, 6 meses, 0 días. Para días se usa año comercial de 360 días.
El programa calcula el tiempo entre las dos fechas: tiempo real (base 365 días) y tiempo aproximado (base 360 días). Si no llenas ambas fechas, se usará el tiempo manual en años/meses/días.

Calculadora de Descuento Simple

Descuento comercial: D = M · d · t, C = M − D.
Descuento real (justo): M = C (1 + d · t), C = M / (1 + d · t), D = M − C.

Elige el tipo de descuento que indique el problema.
Importe del pagaré / documento a su vencimiento.
%
Ej: si dice “18% anual”, escribe 18 y elige “% anual”. Si dice “2% mensual”, escribe 2 y elige “% mensual”.
Llena solo lo que necesites. Ej: 3 meses → 0 años, 3 meses, 0 días. Para días se usa año comercial de 360 días.
